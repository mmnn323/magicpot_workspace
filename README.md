- Server : Apache Tomcat (8.5)
- Server Programming : JSP/Servlet, AJAX
- Framework : Spring, MyBatis
- 프로젝트 관리도구 : Maven
- IDE : Eclipse, STS3, Visual Studio Code, sqldeveloper
- Modeling Tool : StarUML, erdcloud, 카카오오븐
